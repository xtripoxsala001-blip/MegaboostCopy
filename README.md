# MegaboostCopy

## Descrição
MegaboostCopy é um projeto para gerenciar e copiar arquivos com eficiência.

## Funcionalidades
- ✅ Cópia rápida de arquivos
- ✅ Suporte a múltiplos formatos
- ✅ Interface simples e intuitiva

## Instalação
```bash
git clone https://github.com/xtripoxsala001-blip/MegaboostCopy.git
cd MegaboostCopy
```

## Uso
```bash
python megaboost.py
```
